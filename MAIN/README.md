# MAIN Django Project

This is a Django web application developed as part of the PEP Winter Training program.

## Today's Work (Day 10)

- Developed a simple frontend using Django.
- Connected rendering files/templates for displaying content, including base.html, home.html, about.html, and services.html.
- Configured views and URLs for home, about, and services apps.
- Integrated static files like style.css for styling.

## Structure

- **main/**: Main Django settings and configuration (settings.py, urls.py, etc.).
- **home/**: Home app with views.py and urls.py for the homepage.
- **about/**: About app with views and templates.
- **services/**: Services app with views and templates.
- **templates/**: HTML templates (base.html, home.html, about.html, services.html).
- **static/**: Static files including style.css.

## Getting Started

1. Navigate to the MAIN directory.
2. Run `python manage.py runserver` to start the development server.
3. Access the application at `http://127.0.0.1:8000/`.

Ensure Django is installed and configured properly.

# Project2

This is a Django project for PEP Winter Training, featuring a website with Tailwind CSS for styling.

## Setup

1. Install Python dependencies: `pip install -r requirements.txt`
2. Install Node.js dependencies: `npm install`
3. Run Django migrations: `python manage.py migrate`
4. Build Tailwind CSS: `python manage.py tailwind build`
5. Run the development server: `python manage.py runserver`

## Features

- Django backend
- Tailwind CSS for frontend styling
- Custom templates in `template/` directory
- Static files in `static/` directory

## Apps

- yoursite: Main app with views and URLs
- theme: Tailwind CSS integration

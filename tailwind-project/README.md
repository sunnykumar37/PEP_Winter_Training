# tailwind-project (Day 9 Work)

This is a Django project for PEP Winter Training, focusing on integrating Tailwind CSS for modern frontend styling.

## Setup

1. Install Python dependencies: `pip install -r requirements.txt`
2. Install Node.js dependencies: `npm install` (if applicable)
3. Run Django migrations: `python manage.py migrate`
4. Build static files if needed
5. Run the development server: `python manage.py runserver`

## Features

- Django backend
- Tailwind CSS for responsive and modern UI
- Custom templates in `templates/` directory
- Static files in `theme/static/` directory

## Apps

- myapp: Main Django app
- theme: Handles Tailwind CSS integration and static file management
